# Определение оптических постоянных n и k тонких плёнок по спектрам T и R

### В рамках данной работы: 
* **n** — коэффициент преломления тонкой пленки;
* **k** — коэффициент экстинкции тонкой пленки;
* **T** — коэффициент пропускания;
* **R** — коэффициент отражения;
* **λ** — длина волны.

#### Полноценное десктопное приложение `nk-Analyzer.exe` собрано на основе данного консольного скрипта `Diplom.py` с помощью библиотеки **Tkinter** и автосборщика приложений **PyInstaller**, и также находится в репозитории.

---

Данная программа предназначена для расчёта показателя преломления **n** и коэффициента экстинкции **k** тонкой плёнки на прозрачной подложке по экспериментальным спектрам пропускания **T(λ)** и отражения **R(λ)**. Алгоритм использует формулы Френеля для многослойной структуры и комбинацию глобального перебора (для первой точки) с локальным уточнением (для остальных длин волн), что обеспечивает высокую точность и устойчивость к шумам.

### Особенности

* **Двухэтапный алгоритм:** глобальный поиск для начальной точки + локальное уточнение с учётом плавности спектра.  
* **Поддержка данных:** работа с запятой в качестве десятичного разделителя.  
* **Автоматическое определение:** количества экспериментальных точек из первой строки файла.  
* **Визуализация результатов:** четыре информативных графика ($T(\lambda)$, $R(\lambda)$, $n(\lambda)$, $k(\lambda)$).  
* **Лёгкая настройка:** параметров поиска (диапазоны $n$, $k$, шаги, окно локального поиска).

### Требования
* Python 3.6 или выше
* Библиотека **matplotlib** (устанавливается через `pip`)

### Установка и запуск

1. **Скопируйте файлы проекта в одну папку:**
   * `Diplom.py` – основной скрипт
   * `data.txt` – файл с экспериментальными данными (пример прилагается)

2. **Установите зависимость:**
   ```bash
   pip install matplotlib
   ```

3. **Запустите программу:**
   ```bash
   python Diplom.py
   ```

### Формат входных данных

Файл `data.txt` должен содержать:
* **Первая строка** — служебная информация, обязательно содержащая целое число (количество последующих строк с данными).
* **Последующие строки** — четыре числа, разделённых пробелами или табуляцией:
    1. Длина волны **λ (нм)** — десятичное число (запятая как разделитель, например `350,0`).
    2. Показатель преломления подложки **n_sub** (для данной λ) — также с запятой.
    3. Коэффициент пропускания **T** (отн. ед.).
    4. Коэффициент отражения **R** (отн. ед.).

**Пример:**
```text
λ,нм   n       T              R         751
350  1,4768 0,016243547 0,305063025
351  1,4766 0,016769457 0,304887018
...
```

> **Важно:** толщина плёнки жёстко задана в коде (`d_nm = 100.0` нм). При необходимости измените это значение в файле `Diplom.py`.

---

### Алгоритм работы

#### 1. Глобальный перебор (для каждой длины волны)
Для каждой точки **λ** перебираются все возможные пары **(n, k)** в заданных диапазонах:
* **n_min … n_max** (по умолчанию 1.0 … 14.0)
* **k_min … k_max** (по умолчанию 0.0 … 1.0)

Шаги перебора задаются переменными `n_steps`, `k_steps` (по умолчанию 100×100 = 10 000 вариантов). Для каждой пары вычисляются теоретические значения **R_теор** и **T_теор** по формулам Френеля, а затем невязка:
$$F = |R_{теор} − R_{эксп}| + |T_{теор} − T_{эксп}|$$
Выбирается пара с минимальной **F**. Результаты сохраняются для всех точек.

#### 2. Выбор опорной точки
Из всех полученных пар выбирается точка с наименьшей глобальной невязкой — она считается опорной.

#### 3. Локальное уточнение (влево и вправо от опорной)
Для каждой соседней точки поиск ведётся в ограниченном окне вокруг значений **(n, k)** предыдущей уточнённой точки. 
* Размер окна задаётся параметрами `delta_n` и `delta_k` (по умолчанию 0.1). 
* Внутри окна перебор выполняется с более высоким разрешением (`n_steps_local`, `k_steps_local` = 100). 

Это гарантирует плавность спектральных зависимостей **n(λ)** и **k(λ)**.

### Расчёт теоретических R и T
**Формулы Френеля** для системы *воздух – тонкая поглощающая плёнка – прозрачная подложка – воздух* реализованы в функции `compute_RT(...)`. Они учитывают многолучевую интерференцию и поглощение в плёнке.

---

### Выходные данные

#### 1. Консольный вывод:
* Количество прочитанных записей.
* Информация об опорной точке (λ, n, k, невязка).
* Ход уточнения (λ → n, k, F) для всех длин волн.

#### 2. Графики (интерактивное окно):
* Сравнение экспериментального и теоретического пропускания **T(λ)**.
* Сравнение экспериментального и теоретического отражения **R(λ)**.
* Полученная зависимость показателя преломления плёнки **n(λ)**.
* Полученная зависимость коэффициента экстинкции **k(λ)**.

> Массив результатов `final_results` доступен в коде для дальнейшего анализа или сохранения в файл.

---

### Настройка параметров

Все ключевые параметры находятся в основной части скрипта после чтения данных:


| Параметр | Описание | Значение по умолчанию |
| :--- | :--- | :--- |
| **n_min, n_max** | Диапазон поиска n | 1.0, 14.0 |
| **k_min, k_max** | Диапазон поиска k | 0.0, 1.0 |
| **d_nm** | Толщина плёнки (нм) | 100.0 |
| **n_steps** | Число шагов по n и k (глобальный перебор) | 100 |
| **delta_n** | Полуширина окна локального поиска по n | 0.1 |
| **delta_k** | Полуширина окна локального поиска по k | 0.1 |
| **n_steps_local** | Число шагов по n и k (локальный перебор) | 100 |

*При работе с зашумлёнными данными рекомендуется уменьшить `delta_n` и `delta_k`.*
